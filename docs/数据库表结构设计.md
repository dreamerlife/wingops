# 统一运维管理平台数据库表结构设计

## 文档信息

| 项目 | 内容 |
|------|------|
| 文档版本 | v1.0 |
| 更新日期 | 2026-06-11 |
| 当前实现状态 | M0-M2 基本功能已实现；M3-M7 尚未开始开发 |
| 主存储 | PostgreSQL、VictoriaMetrics |

## 1. 数据分层

| 数据类型 | 存储 | 说明 |
|----------|------|------|
| 用户、权限、审计、系统配置 | PostgreSQL | M0-M1 已实现 |
| CMDB 模型、资产、同步密钥 | PostgreSQL | M2 已实现 |
| 监控目标、采集配置、仪表盘配置 | PostgreSQL | M3 设计 |
| 监控指标样本 | VictoriaMetrics | M3 设计，不落 PostgreSQL 明细表 |
| 告警规则、事件、备注、静默 | PostgreSQL | M4 设计 |
| 联系人、通知记录、工单 | PostgreSQL | M5 设计 |
| 异步事件 | NATS | 告警触发、通知分发、工单流程事件 |

## 2. M0-M2 已实现表

### 2.1 用户与认证

#### users

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK, default gen_random_uuid() | 用户 ID |
| username | TEXT | NOT NULL, UNIQUE | 登录名 |
| password_hash | TEXT | NOT NULL | bcrypt 密码哈希 |
| display_name | TEXT | NOT NULL | 显示名称 |
| status | TEXT | NOT NULL, default active | 用户状态 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |
| updated_at | TIMESTAMPTZ | NOT NULL | 更新时间 |

### 2.2 RBAC

#### roles

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 角色 ID |
| name | TEXT | NOT NULL, UNIQUE | 角色编码 |
| display_name | TEXT | NOT NULL | 角色名称 |

#### permissions

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 权限 ID |
| code | TEXT | NOT NULL, UNIQUE | 权限码 |
| description | TEXT | NOT NULL | 权限说明 |

#### user_roles

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| user_id | UUID | PK, FK users(id) | 用户 ID |
| role_id | UUID | PK, FK roles(id) | 角色 ID |

#### role_permissions

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| role_id | UUID | PK, FK roles(id) | 角色 ID |
| permission_id | UUID | PK, FK permissions(id) | 权限 ID |

### 2.3 审计与系统配置

#### audit_logs

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 日志 ID |
| actor_id | UUID | nullable | 操作人 ID |
| method | TEXT | NOT NULL | HTTP 方法 |
| path | TEXT | NOT NULL | 请求路径 |
| status_code | INT | NOT NULL | 响应状态码 |
| resource | TEXT | NOT NULL, default '' | 资源描述 |
| created_at | TIMESTAMPTZ | NOT NULL | 操作时间 |

#### system_configs

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| key | TEXT | PK | 配置键 |
| value | JSONB | NOT NULL | 配置值 |
| updated_at | TIMESTAMPTZ | NOT NULL | 更新时间 |

### 2.4 CMDB 模型

#### model_groups

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 模型组 ID |
| name | TEXT | NOT NULL, UNIQUE | 模型组编码 |
| display_name | TEXT | NOT NULL | 模型组名称 |
| description | TEXT | NOT NULL, default '' | 描述 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |
| updated_at | TIMESTAMPTZ | NOT NULL | 更新时间 |

#### models

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 模型 ID |
| group_id | UUID | FK model_groups(id) | 所属模型组 |
| name | TEXT | NOT NULL | 模型编码 |
| display_name | TEXT | NOT NULL | 模型名称 |
| description | TEXT | NOT NULL, default '' | 描述 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |
| updated_at | TIMESTAMPTZ | NOT NULL | 更新时间 |

唯一约束：`UNIQUE(group_id, name)`。

#### model_fields

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 字段 ID |
| model_id | UUID | FK models(id) | 所属模型 |
| name | TEXT | NOT NULL | 字段编码 |
| display_name | TEXT | NOT NULL | 字段名称 |
| field_type | TEXT | NOT NULL | 字段类型 |
| required | BOOLEAN | NOT NULL, default false | 是否必填 |
| unique_value | BOOLEAN | NOT NULL, default false | 是否唯一 |
| options | JSONB | NOT NULL, default {} | 枚举、正则、范围等配置 |
| sort_order | INT | NOT NULL, default 0 | 排序 |

唯一约束：`UNIQUE(model_id, name)`。

#### model_relations

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 关系 ID |
| source_model_id | UUID | FK models(id) ON DELETE CASCADE | 源模型 |
| target_model_id | UUID | FK models(id) ON DELETE CASCADE | 目标模型 |
| relation_type | TEXT | NOT NULL | 关系类型 |
| display_name | TEXT | NOT NULL | 关系名称 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |

唯一约束：`UNIQUE(source_model_id, target_model_id, relation_type)`。

### 2.5 CMDB 资产

#### asset_groups

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 分组 ID |
| name | TEXT | NOT NULL, UNIQUE | 分组编码 |
| display_name | TEXT | NOT NULL | 分组名称 |
| dimension | TEXT | NOT NULL | 分组维度，如 environment、business、region |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |

#### assets

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 资产 ID |
| model_id | UUID | FK models(id) | 所属模型 |
| unique_key | TEXT | NOT NULL | 模型内唯一键 |
| status | TEXT | NOT NULL, default running | 生命周期状态 |
| attributes | JSONB | NOT NULL, default {} | 动态属性 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |
| updated_at | TIMESTAMPTZ | NOT NULL | 更新时间 |

唯一约束：`UNIQUE(model_id, unique_key)`。

#### asset_group_members

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| asset_id | UUID | PK, FK assets(id) | 资产 ID |
| group_id | UUID | PK, FK asset_groups(id) | 分组 ID |

#### asset_change_logs

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 变更记录 ID |
| asset_id | UUID | FK assets(id) | 资产 ID |
| actor_id | UUID | nullable | 操作人 ID |
| before_value | JSONB | NOT NULL | 变更前数据 |
| after_value | JSONB | NOT NULL | 变更后数据 |
| created_at | TIMESTAMPTZ | NOT NULL | 变更时间 |

## 3. M3 监控系统表设计

#### monitor_targets

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 监控目标 ID |
| asset_id | UUID | FK assets(id) | 关联 CMDB 资产 |
| target_type | TEXT | NOT NULL, default host | 目标类型：host、network、middleware |
| address | TEXT | NOT NULL | 采集地址 |
| status | TEXT | NOT NULL, default active | 目标状态 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |

#### collect_configs

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 配置 ID |
| target_id | UUID | FK monitor_targets(id) | 监控目标 |
| interval_seconds | INT | NOT NULL, default 30 | 采集间隔 |
| enabled | BOOLEAN | NOT NULL, default true | 是否启用 |
| updated_at | TIMESTAMPTZ | NOT NULL | 更新时间 |

#### dashboards

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 仪表盘 ID |
| name | TEXT | NOT NULL | 仪表盘名称 |
| description | TEXT | NOT NULL, default '' | 描述 |
| layout | JSONB | NOT NULL, default [] | 面板布局和查询配置 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |

监控指标明细写入 VictoriaMetrics，统一标签建议包含：`asset_id`、`target_id`、`hostname`、`model`、`environment`。

## 4. M4 告警平台表设计

#### alert_rules

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 规则 ID |
| name | TEXT | NOT NULL | 规则名称 |
| metric | TEXT | NOT NULL | 指标名 |
| operator | TEXT | NOT NULL | 比较运算符 |
| threshold | DOUBLE PRECISION | NOT NULL | 阈值 |
| duration_seconds | INT | NOT NULL, default 300 | 持续时间 |
| severity | TEXT | NOT NULL | P0/P1/P2/P3 |
| asset_group_id | UUID | FK asset_groups(id), nullable | 作用资产分组 |
| enabled | BOOLEAN | NOT NULL, default true | 是否启用 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |
| updated_at | TIMESTAMPTZ | NOT NULL | 更新时间 |

#### alert_events

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 告警事件 ID |
| rule_id | UUID | FK alert_rules(id) | 来源规则 |
| asset_id | UUID | FK assets(id) | 关联资产 |
| metric | TEXT | NOT NULL | 触发指标 |
| value | DOUBLE PRECISION | NOT NULL | 触发值 |
| severity | TEXT | NOT NULL | 告警等级 |
| status | TEXT | NOT NULL | Firing/Acknowledged/Recovered/Closed |
| fingerprint | TEXT | NOT NULL | 收敛指纹 |
| first_fired_at | TIMESTAMPTZ | NOT NULL | 首次触发时间 |
| last_fired_at | TIMESTAMPTZ | NOT NULL | 最近触发时间 |
| recovered_at | TIMESTAMPTZ | nullable | 恢复时间 |
| closed_at | TIMESTAMPTZ | nullable | 关闭时间 |

建议索引：`fingerprint`、`status`、`asset_id`、`last_fired_at`。

#### alert_notes

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 备注 ID |
| event_id | UUID | FK alert_events(id) | 告警事件 |
| actor_id | UUID | FK users(id), nullable | 操作人 |
| note | TEXT | NOT NULL | 备注内容 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |

#### alert_silences

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 静默 ID |
| name | TEXT | NOT NULL | 静默名称 |
| asset_group_id | UUID | FK asset_groups(id), nullable | 资产分组 |
| metric | TEXT | NOT NULL, default '' | 指标名，空表示不限 |
| starts_at | TIMESTAMPTZ | NOT NULL | 开始时间 |
| ends_at | TIMESTAMPTZ | NOT NULL | 结束时间 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |

## 5. M5 联系人与工单流程表设计

#### contacts

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 联系人 ID |
| user_id | UUID | FK users(id), nullable | 关联系统用户 |
| name | TEXT | NOT NULL | 联系人姓名 |
| email | TEXT | NOT NULL, default '' | 默认邮箱 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |

#### notification_channels

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 渠道 ID |
| contact_id | UUID | FK contacts(id) | 联系人 |
| channel_type | TEXT | NOT NULL | email、webhook、sms、phone、wecom、dingtalk |
| address | TEXT | NOT NULL | 渠道地址 |
| enabled | BOOLEAN | NOT NULL, default true | 是否启用 |

#### notification_logs

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 通知记录 ID |
| alert_event_id | UUID | FK alert_events(id), nullable | 关联告警 |
| contact_id | UUID | FK contacts(id), nullable | 接收联系人 |
| channel_type | TEXT | NOT NULL | 渠道类型 |
| content | TEXT | NOT NULL | 通知内容 |
| status | TEXT | NOT NULL | success、failed、pending |
| error_message | TEXT | NOT NULL, default '' | 失败原因 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |

#### work_orders

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 工单 ID |
| alert_event_id | UUID | FK alert_events(id), UNIQUE | 关联告警，确保同一活跃告警不重复建单 |
| asset_id | UUID | FK assets(id) | 关联资产 |
| assignee_id | UUID | FK users(id), nullable | 处理人 |
| status | TEXT | NOT NULL, default pending | pending/processing/completed/closed |
| result | TEXT | NOT NULL, default '' | 处理结果 |
| closed_at | TIMESTAMPTZ | nullable | 闭环时间 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |

建议补充字段：`started_at`、`completed_at`、`updated_at`，用于统计处理时长。

#### work_order_logs

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 记录 ID |
| work_order_id | UUID | FK work_orders(id) | 工单 ID |
| actor_id | UUID | FK users(id), nullable | 操作人 |
| content | TEXT | NOT NULL | 处理记录 |
| created_at | TIMESTAMPTZ | NOT NULL | 创建时间 |

## 6. 核心关系

```text
users ──< user_roles >── roles ──< role_permissions >── permissions

model_groups ──< models ──< model_fields
models ──< assets ──< asset_change_logs
assets ──< asset_group_members >── asset_groups

assets ──< monitor_targets ── collect_configs
assets ──< alert_events >── alert_rules
alert_events ──< alert_notes
alert_events ── work_orders ──< work_order_logs

users ── contacts ──< notification_channels
alert_events ──< notification_logs
```

## 7. 状态枚举建议

| 领域 | 字段 | 值 |
|------|------|----|
| 用户 | users.status | active、disabled |
| API Key | api_keys.status | active、revoked |
| 资产 | assets.status | purchased、racked、running、repairing、retired |
| 监控目标 | monitor_targets.status | active、inactive |
| 告警事件 | alert_events.status | Firing、Acknowledged、Recovered、Closed |
| 通知记录 | notification_logs.status | pending、success、failed |
| 工单 | work_orders.status | pending、processing、completed、closed |
